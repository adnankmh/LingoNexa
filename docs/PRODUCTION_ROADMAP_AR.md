# خارطة الطريق للإطلاق العالمي

## المرحلة 1 — MVP آمن

- اعتماد الاسم التجاري بعد فحص العلامة التجارية والدومين.
- اختيار 3–5 لغات إطلاق بدل نشر 67 لغة بمحتوى ضعيف.
- كتابة A1 وA2 كاملين مع مراجعة لغوية وتسجيل صوت.
- مصادقة ومزامنة وحذف حساب وسياسة خصوصية.
- اختبار مغلق مع 100–300 متعلم.

## المرحلة 2 — المحادثة والذكاء الاصطناعي

- خدمة AI من الخادم مع حماية المفتاح والميزانية.
- معايير تقييم واضحة للنطق والقواعد، وإظهار حدود الذكاء الاصطناعي.
- شات وشركاء مع الحظر والإبلاغ ومكافحة الرسائل المزعجة.
- غرف صوت مع مشرفين وأدوات أمان.

## المرحلة 3 — التوسع التجاري

- اشتراكات شهرية وسنوية وعائلية.
- لوحة CMS سحابية بصلاحيات وأرشفة وإصدارات.
- تحليلات للتعلّم لا للإدمان: إتقان، نسيان، عودة، وإكمال.
- توطين واجهة التطبيق إلى أهم الأسواق.
- إضافة اللغات تدريجيًا بعد اجتياز بوابة الجودة.

## مؤشرات يجب مراقبتها

- إكمال أول درس وأول أسبوع.
- الاحتفاظ D1 / D7 / D30.
- عدد الكلمات النشطة لا الكلمات المعروضة فقط.
- تحسن الاستماع والنطق عبر اختبارات ثابتة.
- البلاغات وزمن استجابة الإشراف.
- تكلفة AI والصوت لكل متعلم نشط.

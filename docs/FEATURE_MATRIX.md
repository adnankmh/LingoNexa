# Product feature synthesis

LingoNexa does not reproduce competitor code, artwork, copy, characters, or proprietary course content. It combines common product patterns into an original information architecture and visual system.

| Product pattern observed | Public example | Original LingoNexa implementation |
| --- | --- | --- |
| Short, structured lessons and practical dialogs | Babbel | Five lesson modes per unit, real-world topic structure |
| Gamified path, XP, streak, stories, practice hub | Duolingo | CEFR journey, per-user XP, level exams, dialogue-story library, fast practice |
| AI role-play and answer guidance | Duolingo Max / Busuu Conversations / Memrise | Nexa Coach scenario shell and secure endpoint boundary |
| Native speech and pronunciation work | Babbel / Mango / Memrise | Explicit locale per language, no English fallback, device TTS/STT availability checks |
| Spaced recall | Mango / Babbel / Memrise | Local review queue and review-ready lesson IDs |
| Native-speaker community corrections | Busuu | Correction cards and community feed concept |
| Partner matching, text/voice, rooms | HelloTalk / Speaky | Partner rail, posts, voice-room prototype, moderation actions |
| Culture integrated with language | Mango | Culture notes inside lessons plus articles |
| Offline learning and progress | Babbel / Mango | Local accounts, isolated progress, offline packs and feature toggle |
| Placement and CEFR organization | Mango / Busuu | Four-step onboarding, starter placement check, and A1–C2 course framework |
| Goal-specific courses | Babbel / Mango | Eight original travel, business, health, academic, kids, life-abroad, exam, and media pathways |
| Phrasebook, grammar, and scripts | Broad category pattern | Searchable phrases, A1–C2 grammar atlas, character maps, and handwriting practice |
| Plans, leagues, and records | Broad category pattern | Weekly study plan, achievement milestones, league table, offline packs, and certificate readiness |
| Correct/incorrect motion feedback | Broad category pattern | Original lightweight Lottie feedback plus system sounds |
| Content operations | Broad category pattern | Role-gated admin studio for learning, voice, XP, accounts, modules, and export |

## Content integrity boundary

The 12 interface/core languages have an expanded aligned concept bank. The
remaining catalog languages expose only their verified starter lexicons until
reviewed course packs are supplied. Unsupported target content never falls back
to English, and speech never silently switches to an English voice.

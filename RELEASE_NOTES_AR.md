# ملاحظات إصدار LingoNexa 1.1.0

تاريخ الحزمة: 18 يوليو 2026

## ما الجديد

- تجربة Onboarding كاملة مع اختيار اللغة، الهدف، المدة اليومية، واختبار تحديد مستوى.
- مركز تعلّم موسّع: عبارات وقاموس، قواعد A1–C2، أبجديات وكتابة، ومسارات متخصصة.
- 8 مسارات للأهداف الواقعية و12 موضوع قواعد و12 فئة عبارات.
- خطة أسبوعية شخصية، نظام إنجازات ودوري، تنزيل 67 حزمة دون إنترنت، وسجل شهادات.
- توسعة الملف الشخصي ولوحة الإدارة وإتاحة إعادة بناء الخطة.
- Workflow واحد صحيح يثبت Flutter ويستخدم `flutter pub get` ويبني APK وAAB والويب.
- تعليمات عربية دقيقة لمستخدمي GitHub Desktop دون الحاجة إلى Android Studio محليًا.

## الأساس الموجود

- 67 لغة، ستة مستويات CEFR، و180 عقدة درس مولّدة لكل لغة.
- سبعة أنماط تمارين، TTS/STT، مراجعة، معلّم سيناريوهات، مجتمع، وثمانية ثيمات.
- تصميم متجاوب للهاتف والويب، RTL، ولوحة إعدادات وإدارة محلية.

## حدود يجب معرفتها

- ميزات المجتمع الحي والذكاء الاصطناعي والمزامنة والشهادات الموثقة تحتاج Backend إنتاج آمن.
- المحتوى المرفق محرك تعليمي وStarter Curriculum واسع، وليس آلاف الساعات المراجعة بشريًا لكل واحدة من 67 لغة.
- AAB المخصص للنشر يحتاج Upload Key خاصًا بالمالك.
- لم يتوفر Flutter SDK داخل بيئة إنشاء الحزمة، لذلك يتولى GitHub Actions التحليل والاختبارات والبناء الفعلي.

ابدأ من `REPLACE_INSTRUCTIONS_AR.md` لمعالجة Workflow، ثم راجع `README_AR.md`.

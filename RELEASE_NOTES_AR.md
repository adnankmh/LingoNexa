# ملاحظات إصدار LingoNexa 1.6.0+11

تاريخ الحزمة: 22 يوليو 2026

## ما الجديد

### Global Learning Hub في 1.6.0

- دمج دوائر الأنشطة الخمسة في بطاقة واحدة واضحة تفتح وحدة متكاملة بدل تشتيت المستخدم بين رموز غير مفسرة.
- نقل الشرح الكامل إلى الصفحة الرئيسية وإضافة خانات مستقلة للكلمات والأمثلة والقواعد والامتحانات والقصص والمترجم.
- إضافة صفحة وحدة كبيرة تشمل الشرح والأهداف و12 عنصر مفردات موثقًا والأمثلة والحوار والأنشطة الخمسة والقصة والامتحان والمترجم والتحكم الصوتي.
- رفع الامتحان من 6 أسئلة إلى 20 سؤالًا متنوعًا، وترجمة واجهة الامتحان والنتيجة إلى حزم الواجهة الـ12.
- إضافة مترجم موثق مع تبديل اللغات والنسخ والنطق والاقتراحات، ورفض عرض ترجمة تخمينية على أنها حقيقة.
- دعم تعداد القرّاء المثبتين لكل لغة واختيار القارئ واللهجة وحفظهما، مع سرعة نطق قابلة للضبط واختبار صوت مباشر.
- رفع البنك المترابط إلى 84 مفهومًا × 12 لغة = 1,008 أزواج موثقة و4,032 تدريبًا.
- رفع القواعد إلى 54 درسًا والمسارات المتخصصة إلى 28 مسارًا، وإضافة 4 حركات Lottie جديدة للاستماع والكلام والسلسلة والتنزيل.
- تفعيل حزم التعلم المحلية، وفحص حزمة صوت اللغة بدل الادعاء بأن الصوت محمّل داخل التطبيق.
- تفعيل نشر المجتمع والإعجاب والحفظ والتعليقات والتصحيحات، وإضافة نموذج تدريب صوتي محلي مع توضيح أن المحادثة بين مستخدمين حقيقيين تحتاج Backend إنتاجيًا.
- تفعيل الدوري الأسبوعي حسب XP الحقيقي، وتطوير الشهادات باسم المستخدم واللغة والمستوى وبلغة واجهة يختارها، مع إمكانية الرجوع إلى أي مستوى ناجح.
- إكمال أوصاف الأزرار ودعم المرور بالماوس والضغط المطوّل وقارئات الشاشة، وعدم إبقاء أزرار ذات استدعاءات فارغة.

### Guidance & Content Expansion في 1.5.0

- إضافة شرح بصري أنيق عند المرور أو الضغط المطوّل لكل زر أيقونة، مترجم إلى لغات الواجهة الـ12 ومدعوم دلاليًا لقارئات الشاشة.
- إصلاح فحص التنسيق بحيث ينجح الأمر نفسه في GitHub مع 0 ملفات متغيرة.
- إضافة مدقق إصدار آلي يمنع الأيقونات غير الموصوفة، والترجمات الناقصة، والمفاهيم المكررة، وتراجع أعداد المحتوى.

- تكبير أيقونة التطبيق وإعادة توزيعها داخل المربع الكامل مع نسخ Android وWeb محسّنة.
- 15 مجالًا في كل مستوى: 90 وحدة و450 درسًا و4,500 خطوة تعليمية لكل لغة.
- كل درس يستخدم عدة عبارات مترابطة بدل إعادة عبارة واحدة في خطواته كلها.
- رفع البنك العالمي المترابط إلى 66 مفهومًا × 12 لغة أساسية = 792 زوجًا، ينتج عنها 3,168 مهمة جمل.
- 34 تصنيفًا و22 مسارًا متخصصًا تشمل الطبيب والمطار والفندق والمطعم والطوارئ والتقنية والعائلة والانتقال وخدمة العملاء والقيادة ورعاية الأطفال.
- تفعيل وحدات المسارات المتخصصة مع الاستماع والتتبع ومهمات محادثة بدل الأزرار غير الفعالة.
- 42 موضوع قواعد من A1 إلى C2، مع شرح متعدد المحاور وقواعد وأخطاء شائعة وبنك أنماط وثمانية تدريبات تفاعلية.
- 108 قصص حوارية لكل لغة، 16 سطرًا لكل قصة، وأسئلة فهم واستنتاج وإعادة سرد.

### إصلاح CI والويب في 1.3.1

- إصلاح اختبارات الواجهة الثلاثة: كان زر الثيم العام خارج `Navigator Overlay`، فأصبح داخل مسار التطبيق مع بقائه ظاهرًا أعلى الشاشات.
- النتيجة المحلية المطابقة لـGitHub: 14 اختبارًا ناجحًا، و`flutter analyze` بلا أي مشكلة أو تحذير أو ملاحظة.
- جعل APK وAAB وWeb مستقلة فعلًا وتبدأ مباشرة عند كل Push إلى `main` بدل تخطيها عند فشل Flutter CI.
- إبقاء النشر في Workflow منفصل وآمن، مع فحص تفعيل GitHub Pages قبل محاولة النشر.
- بناء Flutter Web Release بنجاح لمسار `/LingoNexa/`، وتعطيل فحص Wasm التجريبي غير المطلوب لأن الموقع يستخدم بناء JavaScript المستقر.
- تحديث استخدام `RadioGroup` وخيارات لغة التعرف الصوتي إلى واجهات Flutter والحزم الحديثة لإزالة تحذيرات Deprecated.
- تنسيق جميع ملفات Dart وتحديث مدقق المشروع ليتعامل مع التنسيق متعدد الأسطر.

### دقة اللغة والصوت

- إعادة بناء مصدر العبارات بحيث تبقى الكلمة والترجمة والصورة الدلالية والفئة في سجل واحد؛ لم يعد المحرك يخلط قوائم مستقلة قد تنزلق عن بعضها.
- تمرير لغة واجهة المستخدم إلى كل الشروحات والمعاني والقصص والامتحانات وNexa Live بدل افتراض الإنجليزية دائمًا.
- إضافة رمز صوت BCP-47 صريح لكل واحدة من لغات الكتالوج الـ67.
- فحص وجود صوت TTS ولغة STT على الجهاز قبل التشغيل، ومنع الرجوع الصامت إلى الإنجليزية كليًا.
- عند عدم دعم اللغة يعرض التطبيق رسالة تثبيت صوت اللغة بدل نطق خاطئ.
- اللغات غير الموجودة في البنك الموسع تستخدم Starter Lexicon المتحقق فقط ولا تستعير عبارات إنجليزية.

### الحسابات والتقدم

- إضافة تسجيل دخول وإنشاء مستخدم وتسجيل خروج ودخول ضيف.
- حفظ التقدم وXP والمستوى والهدف والدروس والمراجعات والامتحانات والتنزيلات بصورة منفصلة لكل حساب محلي.
- إضافة حساب المدير المطلوب وحسابي تجربة، مع تخزين بصمات SHA-256 بدل كلمات المرور النصية داخل خدمة المصادقة.
- إضافة شريط مستوى XP ومعلومات الدور والحساب في الملف الشخصي.
- إضافة واجهة Google وFacebook جاهزة للربط، مع توضيح أن التفعيل الحقيقي يحتاج Firebase/OAuth وخيارات المزود.

### التعلم والتقييم

- إضافة وضع Fast learning بست خطوات أساسية لكل درس، مع إبقاء الوضع الكامل ذي عشر خطوات.
- إضافة امتحان مستقل لكل مستوى A1–C2، ودرجة نجاح 70%، ومكافآت XP وفتح المستوى التالي.
- إضافة مكتبة 36 نشاط قصة حوارية موزعة على المستويات لكل مسار لغة، ومبنية من أزواج عبارات متطابقة.
- إضافة شروحات قواعد خاصة باللغات الأساسية الـ12، وأمثلة مترجمة وفق لغة الواجهة.
- تنويع فهرس المفردات بين المستويات بدل إعادة العبارات الأولى نفسها في كل مستوى.
- إضافة رسوم Lottie صحيحة/خاطئة/نجاح وأصوات نظام للتغذية الراجعة.

### الواجهة والإدارة والوسائط

- جعل الإنجليزية واجهة البداية وSnow الفاتح ثيم البداية لأول تشغيل.
- إضافة زر صغير ثابت أعلى كل شاشة يجمع اختيار لغة التعلّم والتبديل بين الثيمات الثمانية.
- إضافة صور دلالية خفيفة بصيغة Emoji للكلمات والعبارات، بحيث لا تزيد حجم الحزمة ولا تتطلب حقوق صور خارجية.
- ضغط الشعار والأيقونات: الشعار أقل من 50KB، والرمز أقل من 20KB، وأيقونات Android لا تتجاوز بضعة كيلوبايت.
- توسيع لوحة المدير للتحكم في التسجيل والقصص والامتحانات والمحادثة والمجتمع والغرف الصوتية وسرعة TTS وXP والوسائط والمحتوى والتصدير.
- حصر فتح لوحة الإدارة وصلاحية حفظ إعداداتها بحساب المدير.

### الاختبارات والتحقق

- إضافة اختبارات لمحاذاة المعاني، منع fallback الإنجليزي، كل رموز الأصوات، الترجمة حسب لغة الواجهة، وحسابات المدير والتجربة.
- توسيع مدقق الحزمة ليتأكد من ضغط الصور، وجود الحسابات والامتحانات والقصص وملفي Lottie، وخمسة Workflows مستقلة فقط.
- حذف `dart.yml` و`build.yml` من الحزمة الجديدة بدل إبقائهما ظاهرين كـWorkflows قديمة؛ أداة Windows تحذف نسخًا قديمة منها عند الاستبدال فوق Repository سابق.

- دمج شعار LingoNexa المرفق داخل التطبيق وإنشاء أيقونات Android وWeb/PWA كاملة.
- إضافة 12 لغة لواجهة التطبيق مع العربية RTL ومبدّل لغة جديد.
- إضافة Nexa Live: محادثة بالصوت والكتابة، شخصيات، سيناريوهات، ترجمة، تصحيح ودرجة طلاقة.
- إضافة مكتبة محتوى عالمية من 444 زوج عبارة مترجم و1,776 مهمة جمل.
- رفع خطوات كل درس ديناميكي من 7 إلى 10، بما يزيد على 120 ألف خطوة عبر 67 لغة.
- إضافة Sentence Lab للبحث والاستماع والتصفية وتتبع الإتقان.

- تقسيم GitHub Actions إلى خمسة Workflows مستقلة: Flutter CI وAPK وAAB وWeb وDeploy GitHub Pages.
- جعل APK وAAB وWeb تبدأ تلقائيًا وبشكل مستقل عند كل Push إلى فرع main.
- فصل بناء Web عن نشر Pages حتى لا يفشل بناء الويب بسبب إعداد Pages.
- إضافة فحص لتفعيل GitHub Pages؛ عند عدم تفعيلها يتم تخطي النشر بأمان بدل خطأ 404.
- ترقية `checkout` إلى v6 و`setup-java` إلى v5 و`upload-artifact` إلى v7 و`upload-pages-artifact` إلى v5 لدعم Node 24.
- إبقاء `deploy-pages@v4` لأنه الإصدار الرسمي الحالي، مع اعتبار تحذير punycode تحذيرًا من الأداة وليس فشلًا في التطبيق.

- ترقية Android Gradle Plugin من 8.7.3 إلى 8.11.1.
- ترقية Kotlin Gradle Plugin من 2.1.0 إلى 2.2.20.
- تثبيت Gradle Wrapper 8.13 وJDK 17 داخل GitHub Actions.
- حذف ملفات Gradle القديمة المتعارضة تلقائيًا أثناء كل بناء.
- إضافة أداة Windows لحذف ملف `dart.yml` القديم ومنع `dart pub get` عند نسخ الإصدار فوق مستودع قديم.
- إضافة ملف Windows ينظف Dart Workflow وملفات Gradle القديمة بنقرة مزدوجة.

- إصلاح RenderFlex overflow في أول شاشة Onboarding على الشاشات القصيرة.
- جعل شاشة الترحيب متجاوبة وقابلة للتمرير مع أحجام مضغوطة تلقائيًا.
- إضافة اختبار واجهة بحجم 800×500 لمنع رجوع مشكلة Overflow.
- توضيح حذف أو تعطيل Dart Workflow القديم من GitHub بدل إعادة تشغيل سجله الفاشل.

- تجربة Onboarding كاملة مع اختيار اللغة، الهدف، المدة اليومية، واختبار تحديد مستوى.
- مركز تعلّم موسّع: عبارات وقاموس، قواعد A1–C2، أبجديات وكتابة، ومسارات متخصصة.
- 8 مسارات للأهداف الواقعية و12 موضوع قواعد و12 فئة عبارات.
- خطة أسبوعية شخصية، نظام إنجازات ودوري، تنزيل 67 حزمة دون إنترنت، وسجل شهادات.
- توسعة الملف الشخصي ولوحة الإدارة وإتاحة إعادة بناء الخطة.
- Workflows مستقلة تثبت Flutter وتستخدم `flutter pub get` وتنتج Artifacts منفصلة.
- تعليمات عربية دقيقة لمستخدمي GitHub Desktop دون الحاجة إلى Android Studio محليًا.

## الأساس الموجود

- 67 لغة، ستة مستويات CEFR، و450 عقدة درس مولّدة لكل لغة.
- ست أو عشر خطوات تدريب في كل درس، TTS/STT صارم، مراجعة، Nexa Live، مجتمع، وثمانية ثيمات.
- تصميم متجاوب للهاتف والويب، RTL، ولوحة إعدادات وإدارة محلية.

## حدود يجب معرفتها

- ميزات المجتمع الحي والذكاء الاصطناعي والمزامنة والشهادات الموثقة تحتاج Backend إنتاج آمن.
- اللغات الأساسية الـ12 لها البنك الموسع؛ اللغات الـ55 الأخرى لها Starter Lexicon فقط وليست آلاف الساعات المراجعة بشريًا.
- تسجيل Google وFacebook لن يعمل إنتاجيًا قبل إضافة إعدادات Firebase/OAuth الخاصة بمالك التطبيق.
- الحسابات المحلية مفيدة للعرض دون خادم، وليست بديلًا عن مصادقة ومزامنة آمنة في الإنتاج.
- AAB المخصص للنشر يحتاج Upload Key خاصًا بالمالك.
- تم التحقق محليًا باستخدام Flutter 3.44.6 وDart 3.12.2 المطابقين لبيئة GitHub Actions، إضافة إلى بقاء GitHub Actions مسؤولًا عن إنتاج ملفات Android النهائية.

ابدأ من `REPLACE_INSTRUCTIONS_AR.md` لمعالجة Workflow، ثم راجع `README_AR.md`.

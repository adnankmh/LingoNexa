# ملاحظات إصدار LingoNexa 1.0.0

تاريخ الحزمة: 18 يوليو 2026

## المنجز

- مشروع Flutter أصلي متعدد المنصات.
- 67 لغة في الكتالوج، واتجاهات كتابة وسكربتات متنوعة.
- مستويات A1–C2، و36 وحدة و180 عقدة درس ديناميكية لكل لغة.
- تمارين اختيار وترتيب واستماع ونطق واستدعاء وبطاقات وثقافة.
- TTS وتعرّف على الكلام، مراجعة محلية، تقدم وXP وسلسلة أيام.
- معلّم سيناريوهات، مجتمع تجريبي، تصحيحات، غرف صوت، وأدوات أمان.
- ثمانية ثيمات، واجهة عربية RTL وإنجليزية، وتصميم متجاوب.
- لوحة إدارة محلية، تصدير إعدادات، ووثائق لبنية CMS مستقبلية.
- Workflow لفحص وبناء APK وAAB والويب ونشر GitHub Pages.
- اختبارات للكتالوج وتوليد الدروس والواجهة الأساسية.

## حدود هذا الإصدار

- المحتوى المرفق Starter Curriculum لإثبات المحرك، وليس دورة بشرية كاملة من A1 إلى C2 لكل لغة.
- المجتمع والغرف والذكاء الاصطناعي يعملون كواجهة وتجربة محلية حتى يتم ربط Backend آمن.
- AAB المخصص للنشر يحتاج مفتاح Upload Key الخاص بالمالك.
- لم يمكن تشغيل Flutter compiler داخل بيئة إنشاء الحزمة لغياب Flutter SDK؛ لذلك يتولى GitHub Actions `flutter analyze` و`flutter test` والبناء بعد الرفع.

ابدأ من `README_AR.md` ثم شغّل `setup_windows.bat` على Windows.
